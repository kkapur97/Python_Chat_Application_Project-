# 💬 Python GUI Chat Application

A real-time multi-client chat application built using Python socket programming and Tkinter GUI.

---

## 🚀 Features

- Real-time messaging
- Multi-client support
- Thread-based communication handling
- GUI interface using Tkinter
- Graceful disconnection using 'bye'

## 🛠️ Tech Stack

- Python
- Socket Programming (TCP)
- Threading
- Tkinter

## 🧠 Concepts Demonstrated

- Client-Server Architecture
- TCP/IP Communication
- Concurrent Programming
- GUI Development

## 📂 Project Structure
server.py
client.py
layout.py
README.md

## ▶️ How to Run

### 1️⃣ Start the Server
python server.py

### 2️⃣ Start the Client (in new terminal)
python client.py

## Exit Chat
Type:bye

to close the connection gracefully.
